# E-Commerce-Dashboard

## Project Title / Headline
📊 E-Commerce Sales Analytics Dashboard
An interactive Power BI dashboard designed to analyze e-commerce sales performance, customer behavior, and business trends.

## Short Description / Purpose
This Power BI dashboard provides insights into sales, profit, quantity sold, and customer purchasing patterns. It helps businesses monitor performance, identify trends, and make data-driven decisions.

## Tech Stack
The dashboard was built using the following tools and technologies:

* 📊 Power BI Desktop – Main platform used for report creation and visualization.
* 📂 Power Query – Used for data cleaning and transformation.
* 🧮 DAX (Data Analysis Expressions) – Used for calculated measures and KPIs.
* 🔗 Data Modeling – Relationships established among tables to enable analysis.
* 📄 File Format – .pbix for development and .png for dashboard preview.

## Data Source
The dataset contains e-commerce sales records including order details, sales amount, profit, quantity sold, customer information, and product categories.

## Features / Highlights

### Business Problem

Businesses often struggle to track sales performance and understand customer purchasing behavior. Without proper analysis, identifying profitable categories and regions becomes difficult.

### Goal of the Dashboard

* Monitor overall sales and profit.
* Analyze category-wise and region-wise performance.
* Identify customer buying trends.
* Support data-driven business decisions.

### Key Visuals

* KPI cards for Total Sales, Profit, and Quantity.
* Category-wise sales analysis.
* Region-wise performance comparison.
* Monthly sales trends.
* Customer and product insights.

### Business Impact & Insights

* Identified top-performing categories and regions.
* Revealed sales trends and customer preferences.
* Helped understand profitability across different segments.
* Enabled better strategic and operational decision-making.
